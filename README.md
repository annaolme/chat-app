# Chat App

Real-time chat application built with React and Socket.io.

## Features

- Real-time messaging
- Multiple chat rooms (General, Random, Tech)
- Online users list
- Typing indicators
- System messages (join/leave)
- Responsive design

## Tech Stack

- **Frontend:** React 17
- **Backend:** Node.js, Express, Socket.io
- **Styling:** CSS

## Setup

### Server
```
cd server
npm install
npm run dev
```

### Client
```
cd client
npm install
npm start
```

Server runs on port 4000, client on port 3000.
